# Sieglinde_2.0 (ROS Noetic)
1. git clone in catkin_ws/src folder
2. in catkin_ws do `rosdep install --from-paths src --ignore-src -r -y`
3. Do `catkin_make`

With `roslaunch sieglinde_new display.launch` you can launch RViz
